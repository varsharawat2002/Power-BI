# Power-BI
SuperStore Sales Analysis Dashboard

Objective: To analyze sales performance, profitability, customer segments, and product categories to identify key trends and actionable insights for improving business strategy and operational efficiency.

Dataset Source: SuperStore Sales Data

Steps involved:

Data Cleaning:
  Imported CSV and transformed data in Power Query,
  Corrected data types for all columns,
  Removed blanks, duplicates, and errors,
  Handled null values and replaced binary codes
  
Data Modeling:
  Created DAX calculations (Average Delivery Days),
  Developed measures for Sales, Profit, Quantity
  
Visualization:
  Built 10+ charts (Bar, Area, Donut, Map, Line),
  Created 4 KPI cards,
  Added interactive slicers
  
Key Insights & Interpretation:
  Regional: West region leads with 33%, California is top state,
  Temporal: Peak sales in Dec/Nov, highest profit in Oct/Dec,
  Customer: Consumer segment contributes 48%,
  Product: Phones, Chairs, Binders are top sellers,
  Shipping: 4-day average delivery maintained,
  Payment: 43% prefer Cash on Delivery,
  Year on Year: March 2020 showed 4x profit increase
